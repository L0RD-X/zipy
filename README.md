# ZiPy
Como o nome sugere dá pra saber oque é,<br>
é um script para compactar e descompactar arquivos em python 
# Modo de Uso
Digite ```./zipy --zipar pasta/``` para compacatar o arquivo
digite apenas ```./zipy``` para obter o help

# Instalação
Nâo precisa, mas se quiser usar sem o ```./``` mova o script para pasta ```/usr/bin```
