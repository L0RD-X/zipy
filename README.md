# ZiPy
Como o nome sugere dá pra saber o'que é. Um script para compactar e descompactar arquivos zip de modo rápido e simples de usar, em python.
 
# Modo de Uso
Para compactar o arquivo ou pasta digite: <br>```./zipy --zipar pasta/``` <br>
Para descompactar um arquivo zip digite: <br>```./zipy --extrair arquivo.zip```<br> Para ver o modo de uso com mais detalhes digite ```./zipy --help``` ou apenas ```./zipy``` para obter o help.

# Instalação
Nâo precisa, mas se quiser usar sem o ```./``` mova o script para a pasta ```bin/```.<br>
Caso esteja no windows, não sei, se vira

# Observações
As permissões ficarão por sua conta.<br>
Se for compactar mais de um arquivo coloque-os em uma pasta
