# ZiPy
Como o nome sugere dá pra saber oque é. Um script para compactar e descompactar arquivos zip de modo rápido e simples de usar em python 
# Modo de Uso
Digite: <br>```./zipy --zipar pasta/``` <br>para compacatar o arquivo
e: <br>```./zipy --extrair arquico.zip```<br> para descompactar <br> digite apenas ```./zipy``` para obter o help

# Instalação
Nâo precisa, mas se quiser usar sem o ```./``` mova o script para pasta ```/usr/bin```
<br>(Ps: as permissões ficarão por sua conta)
